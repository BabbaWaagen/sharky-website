# Component Reference<a name="component-components"></a>

You can use the **Entity Inspector** to add the following components to you entities\. 

**Note**  
Some components are only available through gems\. For example, you must enable the [Rain](/docs/userguide/gems-system-gem-rain.md) gem for your project, so that the **[Rain](/docs/userguide/component-rain.md)** component is available\.  
For more information, see [Add modular features and assets with Gems](/docs/userguide/gems-system-gems.md)\.

## PhysX<a name="component-entity-physx"></a>
