The meshes, textures, and animations you create for your projects, also known as source assets, must be processed by the **Asset Pipeline** before you can use them in your **Open 3D Engine** projects. The tutorials in this section cover all the basics of the tools you will use to customize how runtime optimized product assets are generated with **Scene Settings**.
