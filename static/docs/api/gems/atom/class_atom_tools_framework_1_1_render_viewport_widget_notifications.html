Detailed Description

Provides an interface for providing notifications specific to RenderViewportWidget.

Note
Most behaviors in RenderViewportWidget are handled by its underyling ViewportContext, this bus is specifically for functionality exclusive to the Qt layer provided by RenderViewportWidget.
