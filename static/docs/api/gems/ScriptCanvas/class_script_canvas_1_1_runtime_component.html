Detailed Description

Runtime Component responsible for loading a ScriptCanvas Graph from a runtime asset. It contains none of the Graph functionality of Validating Connections, as well as adding and removing nodes. Furthermore none of the functionality to remove and add variables exist in this component. It is assumed that the graph has runtime graph has already been validated and compiled at this point. This component should only be used at runtime
