Detailed Description

Filter for passes which have a matching name and also with ordered parents. For example, if the filter is initialized with pass name: "ShadowPass1" pass parents names: "MainPipeline", "Shadow" Passes with these names match the filter: "Root.MainPipeline.SwapChainPass.Shadow.ShadowPass1" or "Root.MainPipeline.Shadow.ShadowPass1" or "MainPipeline.Shadow.Group1.ShadowPass1"

Passes with these names wont match: "MainPipeline.ShadowPass1" or "Shadow.MainPipeline.ShadowPass1"
