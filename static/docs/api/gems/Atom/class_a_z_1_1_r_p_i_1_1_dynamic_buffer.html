Detailed Description

A DynamicBuffer represents a transient GPU buffer which only be valid for one frame after it's acquired. The acquired DynamicBuffers become invalid when DynamicDrawInterface::Get()->FrameEnd() is called. DynamicBuffers are allocated by DynamicBufferAllocator. Check the description of DynamicBufferAllocator class for detail. The typical usage: // For every frame auto buffer = DynamicDrawInterface::Get()->GetDynamicBuffer(size); if (buffer) // the buffer could be empty if the allocation failed.e { // write data to the buffer buffer->Write(data, size); // Use the buffer view for DrawItem or etc. } Note: DynamicBuffer should only be used for InputAssembly buffer or Constant buffer (not supported yet).
