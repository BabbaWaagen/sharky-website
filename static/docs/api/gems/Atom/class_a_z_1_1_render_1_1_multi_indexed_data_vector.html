Detailed Description

template<typename ... Ts>
class AZ::Render::MultiIndexedDataVector< Ts >

MultiIndexedDataVector is similar to IndexedDataVector but adds support for multiple different data vectors each containing different types i.e. structure of (N) arrays See IndexedDataVectorTests.cpp for examples of use
