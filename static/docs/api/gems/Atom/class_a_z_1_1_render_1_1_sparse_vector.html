Detailed Description

template<typename T>
class AZ::Render::SparseVector< T >

SparseVector stores elements in a vector under the hood, but allows for empty slots in the vector. This means that elements can be added or removed without affecting the location of other elements. When a new element is reserved in the SparseVector, it will use a free slot if available. Otherwise, it will push the data onto the end of the vector.

This class works by storing a linked list of elements in the empty slot, as well as an index to the first empty slot. Since the linked list uses size_t for its type, this class must be templated to a class that is at least sizeof(size_t).
