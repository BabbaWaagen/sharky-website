Detailed Description

Handler for a list of ExecuteGroups that are part of the same graph group. All the execute groups share the same renderpass and each of the groups correspond to a subpass of the renderpass. Each execute group uses one or more secondary command list. The handler owns the primary command list that will execute the secondary command list of each execute group.
