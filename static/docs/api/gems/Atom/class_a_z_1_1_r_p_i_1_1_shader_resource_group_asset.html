Detailed Description

This asset defines the layout of a shader resource group, including any relevant metadata about reflected shader inputs.

This is an immutable, serialized asset. It can be either serialized-in or created dynamically using ShaderResourceGroupAssetCreator. See RPI::ShaderResourceGroup for runtime features based on this asset.
