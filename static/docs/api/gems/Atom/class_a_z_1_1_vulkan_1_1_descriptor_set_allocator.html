Detailed Description

Allocator for creating descriptor sets. Each descriptor set is allocated from a descriptor set pool. When the pool can't allocate more descriptor sets we create a new pool. We use the return value from Vulkan to check if the pool ran out of memory and we need to create a new one. A DescriptorSetAllocator is used to generate new descriptor set pools when needed.
