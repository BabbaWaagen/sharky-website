Checks a ray for intersection against this model. The ray must be in the same coordinate space as the model. Important: only to be used in the Editor, it may kick off a job to calculate spatial information. [GFX TODO][ATOM-4343 Bake mesh spatial during AP processing]

Parameters
rayStartposition where the ray starts
dirdirection where the ray ends (does not have to be unit length)
distanceFactorif an intersection is detected, this will be set such that distanceFactor * dir.length == distance to intersection
normalif an intersection is detected, this will be set to the normal at the point of intersection
Returns
true if the ray intersects the mesh
