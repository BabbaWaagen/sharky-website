Detailed Description

template<typename SubAllocator, typename View>
class AZ::Vulkan::MemoryTypeAllocator< SubAllocator, View >

This class is a wrapper over the "SubAllocator" class. Besides of exposing the "SubAllocator" allocation functions (alloc, dealloc, etc), it provides the functionality of allocating memory objects that are bigger than the "SubAllocator" page (a "unique" allocation), it creates and holds the PageAllocator object, it provides thread safe access to the "SubAllocator" and it wraps allocations in a "View" object. Allocations are thread safe.
