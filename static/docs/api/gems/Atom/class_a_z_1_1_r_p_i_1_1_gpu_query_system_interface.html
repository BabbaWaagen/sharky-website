Detailed Description

The GpuQuerySystem is the interface the user communicates with in order to create RPI Queries. For each QueryType, the GpuQuerySystem creates a RPI QueryPool. The GpuQuerySystem will create a RPI Query instance in the applicable RPI QueryPool depending on the requested QueryType.
