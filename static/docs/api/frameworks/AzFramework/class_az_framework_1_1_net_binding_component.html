Detailed Description

NetBindingComponent enables network synchronization for the entity. It works in conjunction with NetBindingComponentChunk and NetBindingSystemComponent to perform network binding and notifies other components on the entity to bind their ReplicaChunks via the NetBindable interface.

Entities bound to proxy replicas will be automatically destroyed when they are unbound from the network.
