template<class InputDeviceType >
virtual void AzFramework::InputDeviceImplementationRequest< InputDeviceType >::CreateCustomImplementation ( CreateFunctionType  createFunction)
pure virtual

Create a custom implementation for all the existing instances of this input device type. Passing InputDeviceType::Implementation::Create as the argument will create the default device implementation, while passing nullptr will delete any existing implementation.
