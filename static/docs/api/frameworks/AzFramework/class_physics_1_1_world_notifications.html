Detailed Description

Broadcasts notifications for a specific Physics::World. This bus is addressed on the id of the world. Subscribe to the bus using Physics::DefaultPhysicsWorldId for the default world, or Physics::EditorPhysicsWorldId for the editor world.

◆ OnPrePhysicsTick()

virtual void Physics::WorldNotifications::OnPrePhysicsTick ( [[maybe_unused] ] float  deltaTime)
inlinevirtual

Broadcast before each world simulation tick. Each tick may include multiple fixed timestep subticks. For example, if the fixed timestep was set at 10ms, and a game tick of 25ms occurred, 2 fixed timestep subticks would be performed this tick, and the remaining 5ms would be accumulated for the subsequent tick. So, in this example, the events fired would be: OnPrePhysicsTick (for the whole 25ms tick) OnPrePhysicsSubtick (for the first 10ms subtick) OnPostPhysicsSubtick (for the first 10ms subtick) OnPrePhysicsSubtick (for the second 10ms subtick) OnPostPhysicsSubtick (for the second 10ms subtick) OnPostPhysicsTick (for the whole 25ms tick)

Parameters
deltaTimeThe duration of the tick as a whole (which may contain multiple fixed timestep subticks).
