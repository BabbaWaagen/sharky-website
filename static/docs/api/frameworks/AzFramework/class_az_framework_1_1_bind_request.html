enum AzFramework::BindRequest::State : AZ::u8

Represents the state of this bind request and it's relation to the slice instantiation process.

Enumerator
FirstBindInSlice 

This is the first request that led to instantiating a slice.

PlaceholderBind 

The request is a placeholder in case a real bind request arrives later. Some part of the slice may never be bound (e.g. if a replica is omitted by Interest Manager)

LateBind 

The real request did arrive to replace a placeholder request.
