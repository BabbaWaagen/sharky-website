Detailed Description

The class is used to store a MaterialLibraryAsset and a vector of MaterialIds selected from the library

This class is used to store a reference to the library asset and user's selection of the materials from this library.
It also reflects UI controls for assigning MaterialLibraryAsset and selecting a material from it. You can reflect this class in EditorContext to provide UI for selecting materials on any custom component or QWidget.
