◆ SetTriggerEventCallback()

virtual void Physics::World::SetTriggerEventCallback ( ITriggerEventCallback *  triggerCallback)
pure virtual

Similar to SetEventHandler, relevant for Touch Bending.

SetEventHandler is useful to catch onTrigger events when the bodies involved were created with the standard physics Components attached to entities. On the other hand, this method was added since Touch Bending, and it is useful for the touch bending simulator to catch onTrigger events of Actors that don't have valid AZ:EntityId.
