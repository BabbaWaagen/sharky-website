virtual AZ::u32 Physics::PhysicsMaterialRequests::GetFirstSelectedMaterialIndex ( const MaterialSelection &  materialSelection)
pure virtual

Returns index of the first selected material in MaterialSelection's material library. A MaterialSelection can contain multiple material selections. Returned index is 0-based where 0 is the Default material, and materials from the material library are 1 and onwards.
