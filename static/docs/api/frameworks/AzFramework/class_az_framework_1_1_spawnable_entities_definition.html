Detailed Description

Interface definition to (de)spawn entities from a spawnable into the game world. While the callbacks of the individual calls are being processed they will block processing any other request. Callbacks can be issued from threads other than the one that issued the call, including the main thread. Calls on the same ticket are guaranteed to be executed in the order they are issued. Note that when issuing requests from multiple threads on the same ticket the order in which the requests are assigned to the ticket is not guaranteed.
