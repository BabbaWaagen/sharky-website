JsonSerializationResult::JsonIssueCallback AZ::JsonApplyPatchSettings::m_reporting

Optional callback when issues are encountered. If not provided reporting will be forwarded to the default issue reporting. This can also be used to change the returned result code to alter the behavior of the patch application process.
