Detailed Description

Best fit allocation scheme using external map. External map allows us to use this allocator with uncached memory, because the tracking node is stored outside the main chunk.

void AZ::BestFitExternalMapSchema::GarbageCollect ( )

Since we don't consolidate chucnks at free time (too expensive) we will do it as we need or when we can't allocate memory. This function is at least O(nlogn) where 'n' are the free chunks.
