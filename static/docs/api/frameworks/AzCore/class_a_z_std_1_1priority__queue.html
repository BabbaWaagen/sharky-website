Detailed Description

template<class T, class Container = AZStd::vector<T>, class Predicate = AZStd::less<typename Container::value_type>>
class AZStd::priority_queue< T, Container, Predicate >

Priority queue is complaint with CStd (23.2.3.2) The only extension we have is that we allow access to the underlying container via: get_container function. Check the priority_queue AZStdExamples.
