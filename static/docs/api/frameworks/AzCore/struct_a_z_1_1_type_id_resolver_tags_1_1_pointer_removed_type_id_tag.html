AZ::TypeIdResolverTags::PointerRemovedTypeIdTag Struct Reference

#include <TypeInfo.h>

Detailed Description

PointerRemovedTypeId is used to lookup TypeIds of template specializations created before the pointer typeid typeid was taken into account when generating a templates typeid. Previously an AZStd::vector<AZ::Entity> and AZStd::vector<AZ::Entity*> had the same typeid which causes incorrect Serialization data to be generated for one of the types. For example the class element for the AZStd::vector<AZ::Entity*> should have the ClassElement::FLG_POINTER set for it's elements, but would not if it was registered with the SerailizeContext last.
