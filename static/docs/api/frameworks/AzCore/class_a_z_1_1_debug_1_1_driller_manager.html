Detailed Description

Driller manager will manage all active driller sessions and driller factories. Generally you will never need more than one driller manger. IMPORTANT: Driller systems works OUTSIDE engine systems, you should NOT use SystemAllocator or any other engine systems as they might be drilled or not available at the moment.
