Detailed Description

template<class DrillerContainer>
class AZ::Debug::DrillerRootHandler< DrillerContainer >

Use this class a input parameter to DrillerSAXParserHandler::DrillerSAXParserHandler(). It will handle all root level tags for a standard driller input stream stream.

DrillerContainer should comply to the following requirements:

  • default constructible
  • has a static const bool s_isWarnOnMissingDrillers member to indicate if you want to trigger a warning when a driller is not found in the class.
  • implement a function DrillerHandlerParser* DrillerContainer::FindDrillerHandler(u32 drillerName)
