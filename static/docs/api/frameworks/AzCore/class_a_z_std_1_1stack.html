Detailed Description

template<class T, class Container = AZStd::deque<T>>
class AZStd::stack< T, Container >

Stack container is complaint with CStd (23.2.3.3) The only extension we have is that we allow access to the underlying container via: get_container function. Check the stack AZStdExamples.
