virtual AZ::Vector3 AzToolsFramework::BoxManipulatorRequests::GetBoxScale ( )
pure virtual

Get the scale currently applied to the box. With the Box Shape, the largest x/y/z component is taken so scale is always uniform, with colliders the scale may be different per component.

◆ GetCurrentTransform()

virtual AZ::Transform AzToolsFramework::BoxManipulatorRequests::GetCurrentTransform ( )
pure virtual

Get the transform of the box shape/collider. This is used by BoxComponentMode instead of the \AZ::TransformBus because a collider may have an additional translation/orientation offset from the Entity transform.
