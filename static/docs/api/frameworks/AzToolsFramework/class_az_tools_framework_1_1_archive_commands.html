Detailed Description

ArchiveCommands This bus handles messages relating to archive commands archive commands are ASYNCHRONOUS archive formats officially supported are .zip do not block the main thread waiting for a response, it is not okay you will not get a message delivered unless you tick the tickbus anyway!
