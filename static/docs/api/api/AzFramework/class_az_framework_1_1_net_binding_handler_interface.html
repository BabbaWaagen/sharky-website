Detailed Description

The NetBindingSystemComponent notifies net binding handlers of binding events on this bus. The net binding component implements this interface and listens on the NetBindingHandlerBus.

virtual void AzFramework::NetBindingHandlerInterface::BindToNetwork ( GridMate::ReplicaPtr  bindTo)

Called to let the entity know that it should bind to the network. If bindTo is set, it means that the entity is a proxy and the handler should bind the entity to the specified replica, otherwise it should bind to a new replica and add it via NetBindingSystemBus::AddReplicaMaster.
